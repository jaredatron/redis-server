# Redis::Server

starts a redis server

## Installation

Add this line to your application's Gemfile:

    gem 'redis-server'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install redis-server

## Usage

require 'redis/server'
s = Redis::Server.new
s.start
s.connect.set(:foo,:bar)
